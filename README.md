# nvim_lua_config
my lua config

## Requirements

- Neovim >= 0.5

## Installation

### Setup configuration

```
sh setup.sh
```
## Uninstallation

```
sh remove.sh
```

### Pre-commit

Install [pre-commit](https://pre-commit.com/) and run the following to enable
automatic Stylua checks when committing:

```
pre-commit install
```
