# nvim_lua_config
my lua config

## Requirements

- Neovim >= 0.5

## Installation

### Setup configuration

```
sh setup.sh
```

### Install dependencies

- Please install [packer.nvim](https://github.com/wbthomason/packer.nvim)
- Launch Neovim and execute bellow command.

```
:PackerInstall
:PackerCompile
```

Enjoy!

## Uninstallation

```
sh remove.sh
```
